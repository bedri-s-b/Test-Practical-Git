JavaScript: Array flatten

Кратко обяснение за бърз достъп

Flatten nested arrays one level or completely, depending on the approach.

javascript arrays snippet

Пример

// shallow flatten (one level)
const a = [1, [2, 3], [4, [5]]];
const flatOne = a.flat(); // [1,2,3,4,[5]]

// deep flatten
const flatDeep = a.flat(Infinity); // [1,2,3,4,5]
